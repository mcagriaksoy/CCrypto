Encrypts data using elliptic curve cryptography (ECC).

This function encrypts the given data using the provided public key and the AES-256-CBC encryption algorithm. The encrypted data is stored in the output buffer, and the length of the encrypted data is stored in the output length variable.

Parameters
public_keyThe public key to use for encryption.
plain_textThe data to encrypt.
data_lengthThe length of the data to encrypt.
encrypted_dataThe output buffer to store the encrypted data.
encrypted_data_lengthA pointer to a variable to store the length of the encrypted data.
Returns
A ccrypto_error_type indicating the success or failure of the encryption. If the encryption was successful, CCRYPTO_SUCCESS is returned. If an error occurred during encryption, an appropriate error code is returned.
